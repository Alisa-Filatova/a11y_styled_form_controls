Styled File Uploads

Last updated: July 23, 2018

Cross-browser styling for the native HTML file input type.

Pattern Demo

Pattern Details

Pattern Markup 
<div class="file-up" data-file-input>
  <label class="file-up__label" for="file_upload">
    <span class="file-up__label__text">
      Upload
    </span>
    <input type="file" id="file_upload">
  </label>
</div>

The containing div[data-file-input] provides the necessary selector hook for the JavaScript.
